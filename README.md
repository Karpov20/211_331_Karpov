# 211_331_Karpov

## Описание
Приложение Qt отображает журнал отгрузок в адаптивной сетке `QGridLayout`, контролирует целостность данных и пересчитывает цепочку MD5-хешей для каждого товара. Файлы транзакций загружаются из JSON либо зашифрованных AES-256-CBC блоков (хранятся в Base64); при несоответствии вычисленного и сохранённого хеша текущая и последующие записи подсвечиваются красным цветом, что упрощает поиск ошибок. Интерфейс поддерживает любое количество строк и корректно масштабируется при изменении размеров окна.

## Сборка
1. `cmake -S . -B build`
2. `cmake --build build`
3. Запустите `build/ShipmentLedger.exe`

Требуется установленный Qt 6 (модуль Widgets) и компилятор с поддержкой C++17.

## Работа с данными
- При старте используется файл `data/transactions_valid.json.enc`.
- Кнопка «Открыть» принимает как обычные `.json`, так и зашифрованные `.enc`.
- Для демонстрации ошибок предусмотрен `data/transactions_corrupted.json.enc`, где цепочка нарушена с третьей записи.

## Шифрование файлов
Тестовые наборы зашифрованы ключом AES-256 `ab9f5f69737f3f02f1e2a6d17305eae239f2bba9d6a8ed5e322ad87d3654c9d8` и IV `1af38c2dc2b96ffdd86694092341bc04`. При необходимости можно повторно зашифровать JSON с помощью PowerShell (пример скрипта находится в `data/encrypt.ps1`) либо сторонней утилиты, задав режим CBC и кодировку UTF-8.
